# 스마트 환경에서의 이상행위와 사이버 공격에 대한 모니터링 및 대응 솔루션 제작 (Web)

모니터링 시각화 웹 With ReactJS + Nginx 👁

## 구성 언어 및 라이브러리

-   Javascript
-   ReactJS (웹 프론트)
-   nginx (서버 어플리케이션)

## 설치 방법

### 우선 git Clone

```sh
$ git clone https://github.com/BoB10-SEDR/Web.git
$ cd Web
```

### 초기 패키지 설치

```sh
$ # 의존성 패키지들을 모두 다운받는 과정
$ npm install
```

### 로컬호스트 시작

```sh
$ npm start
```
